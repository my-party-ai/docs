FastParty is available as an official connector in the Claude Connector Store. Once connected, you can create events, invite guests, manage agendas, and more — all through conversation with Claude, without opening the FastParty app.

## What is this?

FastParty connects to Claude through a technology called MCP (Model Context Protocol) — a standard way for apps to give Claude the ability to take actions on your behalf. FastParty's MCP server is hosted at `mcp.fastparty.ai` and powers the integration automatically once you connect.

You don't need to understand any of that to use it. Connect once, and then just ask Claude to do things.

## Who is this for?

- **Existing FastParty users** who also use Claude and want to manage everything from one place
- **Claude power users** who prefer doing everything through AI conversation
- **Teams using Claude for productivity** who want event planning built in alongside their other tools

## How it compares to Antsy

FastParty has two AI experiences: **Antsy**, which lives inside the FastParty app, and the **Claude integration**, which lives inside Claude. They have the same capabilities — the difference is where you use them.

| | Antsy (in FastParty app) | FastParty in Claude |
|---|---|---|
| Where you access it | FastParty app (iOS, Android, web) | Claude (claude.ai or Claude app) |
| Requires FastParty app | Yes | No |
| Requires Claude account | No | Yes |
| Best for | App-first users | Claude-first users |
| Capabilities | Full FastParty feature set | Full FastParty feature set |
