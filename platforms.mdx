FastParty works wherever you are. Use the web app from any browser, the mobile app on your phone, or manage your events directly from Claude — no app required.

## Web app

Go to [app.fastparty.ai](https://app.fastparty.ai) in any modern browser. No installation needed. The full FastParty feature set is available on web, including Antsy, the AI concierge.

## Claude AI

FastParty is available as an official connector in the Claude Connector Store. Once connected, you can create events, invite guests, manage agendas, and more — directly from Claude, without opening the FastParty app.

## Everything stays in sync

All platforms share the same account. An event you create on your phone is instantly available on the web app and in Claude. RSVP updates, potluck sign-ups, and task changes appear everywhere in real time.
